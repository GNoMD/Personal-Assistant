## 一、米诺地尔基本介绍与药理机制

### 1.1 药物基本概况

| 项目 | 内容 |
|------|------|
| **通用名** | 米诺地尔（Minoxidil） |
| **化学名** | 2,4-嘧啶二胺，6-(1-哌啶基)-，3-氧化物 |
| **药物类别** | 周围血管舒张药；局部外用时为促毛发生长剂 |
| **研发背景** | 1970年代作为口服强效血管扩张剂用于难治性高血压；1987年开发局部外用制剂用于脱发治疗 |
| **国内规格** | 主要为 **2%** 与 **5%** 溶液/搽剂/酊剂；另有泡沫剂、喷雾剂等 |
| **国内批准适应症** | 男性型脱发（雄激素性脱发，AGA）、斑秃（以各产品 NMPA 批准说明书为准） |
| **FDA 批准适应症** | 雄激素性脱发（AGA），适用于 **18岁及以上** 男性和女性 |

**信息来源**：StatPearls（NCBI Bookshelf）；NMPA 批准说明书（如蔓迪、达霏欣等）；Mayo Clinic 药物专论

---

### 1.2 核心适用范围

#### ✅ 合规获批/指南推荐的主要适应症

| 适应症 | 证据等级 | 说明 |
|--------|----------|------|
| **雄激素性脱发（AGA）** | 强推荐（1级，A级证据） | 男女均可，为国内外一线外用药物 |
| **斑秃** | 说明书批准/超说明书有研究支持 | 可单用或联合糖皮质激素等 |
| **女性型脱发（FPHL）** | 强推荐 | 优先 2% 或 5% 泡沫剂（每日1次） |

#### ⚠️ 临床探索性/超说明书应用（需医生评估）

- 化疗相关脱发
- 毛发移植围术期（移植前通常需停药约3天以防出血增加）
- 瘢痕性脱发早期（抗纤维化潜力）
- 牵引性脱发、先天性毛发稀少等

**信息来源**：《2023中国临床实践指南：雄激素性脱发诊断与治疗》；StatPearls（NCBI Bookshelf, PMID 相关条目）

---

### 1.3 不适用或需排除的情况

- 突然发生的、原因不明的或斑片状脱发（需先排除其他病因）
- 产后脱发（通常自限，不宜盲目用药）
- 头皮感染、炎症、破损、疼痛或/sunburn 区域
- 瘢痕性脱发（毛囊已破坏，米诺地尔无效）
- 18岁以下儿童（国内说明书多标注慎用/禁用）
- 孕妇及哺乳期妇女
- 对米诺地尔或辅料（如丙二醇）过敏者

**信息来源**：StatPearls；Mayo Clinic；国内 5% 米诺地尔酊说明书（蔓迪）

---

### 1.4 药理作用机制

米诺地尔促毛发的**确切机制尚未完全阐明**，目前认为通过多途径协同发挥作用：

```
┌─────────────────────────────────────────────────────────────┐
│                    米诺地尔作用机制（简化）                    │
├─────────────────────────────────────────────────────────────┤
│  ① 生物活化：头皮磺基转移酶（SULT）将米诺地尔转化为            │
│     活性形式「米诺地尔硫酸盐」→ 个体酶活性差异影响疗效          │
│  ② 缩短休止期（Telogen）→ 促使毛囊提前进入生长期（Anagen）     │
│  ③ 延长生长期 → 毛发长度、直径增加                             │
│  ④ 血管扩张：开放 ATP 敏感性钾通道，改善毛囊周围微循环           │
│  ⑤ 促进 VEGF 表达 → 增加毛囊周围血管化                         │
│  ⑥ 激活 Wnt/β-catenin 信号通路 → 促进毛囊基质细胞增殖           │
│  ⑦ 抗雄激素效应（部分研究）/ 抗纤维化（瘢痕性脱发探索）          │
└─────────────────────────────────────────────────────────────┘
```

**关键药代动力学数据**：

| 参数 | 数据 |
|------|------|
| 局部经皮吸收率 | 约 **1.4%** 进入系统循环 |
| 头皮吸收动力学 | 约 **1小时** 吸收 50%，**4小时** 吸收 75% |
| 初期见效时间 | 约 **8周** 可见初步变化 |
| 最大效果显现 | 约 **4个月** 达到阶段性峰值，**12～18个月** 趋于稳定 |

**信息来源**：StatPearls（Patel P, Nessel TA, Dinesh Kumar D. Minoxidil. 2023）；Gupta AK, et al. Minoxidil and its use in hair disorders: a review. Dermatol Ther. 2019

## 二、临床疗效数据

### 2.1 男性雄激素性脱发（AGA）

#### 里程碑 RCT：Olsen 等，2002（J Am Acad Dermatol）

| 指标 | 5% 溶液 | 2% 溶液 | 安慰剂 |
|------|---------|---------|--------|
| 研究设计 | 48周、双盲、多中心 RCT，393例男性（18～49岁） | 同左 | 同左 |
| 目标区非毳毛计数增加 | **+18.6 根/cm²** | +12.7 根/cm² | +3.9 根/cm² |
| 5% vs 2% 相对增效 | **多 45%** 毛发再生 | 基准 | — |
| 起效速度 | **更早**出现计数改善 | 较慢 | — |
| 6项疗效指标中优于对照 | 6/6（vs 安慰剂）；4/6（vs 2%） | 优于安慰剂 | — |
| 局部不良反应 | 瘙痒、刺激发生率**更高** | 较低 | 最低 |

**信息来源**：Olsen EA, et al. A randomized clinical trial of 5% topical minoxidil versus 2% topical minoxidil and placebo in the treatment of androgenetic alopecia in men. J Am Acad Dermatol. 2002;47(3):377-385.

#### 中国指南系统评价汇总（男性）

- 纳入 **13项 RCT**（2732例）
- 米诺地尔溶液 vs 安慰剂：毛发计数增加 **MD = 18.36 根/cm²**（95%CI 10.79～25.93）
- **5% 优于 2%**；毛发再生在 **约1年** 达到峰值，随后增速放缓

**信息来源**：《2023中国临床实践指南：雄激素性脱发诊断与治疗》

---

### 2.2 女性雄激素性脱发（FPHL）

| 指标 | 数据 |
|------|------|
| 中国指南系统评价（6项研究，1538例） | 2%/5% 溶液均有效；毛发计数增加 **MD = 13.52 根/cm²**（95%CI 4.81～22.23） |
| Cochrane 系统评价（2017更新） | 2% 米诺地尔：中-重度再生比例 **RR = 1.93**（95%CI 1.51～2.47）vs 安慰剂；总毛发计数增加 **13.18 根/cm²** |
| Meta 分析（女性 2%） | vs 安慰剂：平均差 **12.41 根/cm²** |
| 5% vs 2%（女性） | 疗效差异 **无统计学显著性**（Cochrane）；但 5% 局部刺激、多毛症风险更高 |
| 5% 泡沫剂（每日1次） | 疗效 **优于** 2% 溶液每日2次（MD = 1.60 根/cm²，P = 0.011）；局部瘙痒/皮屑发生率 **更低** |
| 无应答比例 | 约 **40%** 女性患者无明显改善（需联合或换方案） |

**信息来源**：《2023中国临床实践指南》；Cochrane CD007628；Gupta AK, 2019；FPHL therapeutic update review, 2023

### 2.4 见效周期与疗效维持（核心时间轴）

```
时间轴（典型规律，个体差异大）
──────────────────────────────────────────────────────────────
  0～2周     │ 可能无明显变化
  2～6周     │ ⚠「狂脱期」：休止期毛发同步脱落（见下文说明）
  8周        │ 部分患者可见初步改善（StatPearls）
  4个月      │ 多数研究最早观察到明显改善（中国指南）
  4～8个月   │ 可见肉眼可辨的头发生长（国内说明书）
  12～18个月 │ 疗效趋于稳定峰值
  1～2年     │ 中国指南：绝大多数患者疗效接近/达到峰值
  持续使用    │ 需长期维持；停药后数月内疗效逐渐丧失
──────────────────────────────────────────────────────────────
```

#### 关于「狂脱期」（Telogen Effluvium）

- **性质**：米诺地尔缩短休止期，使大量处于休止期的毛发**同步进入脱落→再生周期**，属常见生理现象，**不等于治疗失败**
- **出现时间**：用药后 **2～6周** 较常见
- **持续时间**：通常 **数周内** 自行缓解
- **处理原则**：若无严重头皮刺激，**不建议因此自行停药**；持续恶化超过2周应就医评估

**信息来源**：StatPearls；中华医学会科普文章；Mayo Clinic（用药后2周内脱发可能持续）

---

### 2.5 影响疗效的关键因素

| 因素 | 影响 |
|------|------|
| 脱发病程 | 起病 **5年内**、年轻患者、毛囊未严重.miniaturization 者反应更好 |
| 磺基转移酶活性 | 个体酶活性差异导致疗效不一致（约40%患者反应有限） |
| 用药依从性 | 每日规律用药是疗效基础；漏用频繁显著影响结果 |
| 浓度选择 | 男性 5% 整体优于 2%；女性需平衡疗效与多毛/刺激风险 |
| 联合治疗 | 联合非那雄胺、微针、LLLT 等优于单药（见第四节） |

**信息来源**：StatPearls；Olsen 2002；《2023中国临床实践指南》

## 三、标准化使用方法

### 3.1 不同浓度与剂型的标准剂量对照表

| 剂型/浓度 | 适用人群 | 单次剂量 | 每日频次 | 每日总量上限 | 活性药物量/次 |
|-----------|----------|----------|----------|--------------|---------------|
| **2% 溶液/搽剂** | 女性首选；男性可用 | **1 mL**（约7喷） | **2次/日** | **≤ 2 mL** | 20 mg/次 |
| **5% 溶液/酊剂** | 男性首选 | **1 mL**（约7喷） | **2次/日** | **≤ 2 mL** | 50 mg/次 |
| **5% 溶液** | 女性（医生指导下） | **1 mL** | **1次/日** | **≤ 1 mL** | 50 mg/次 |
| **5% 泡沫剂** | 男性 | **半瓶盖** | **2次/日** | 按说明书 | — |
| **5% 泡沫剂** | 女性 | **半瓶盖** | **1次/日** | 按说明书 | — |

> **⚠ 每日最大安全用量**：溶液剂型 **每日总量不超过 2 mL**（国内说明书统一规定）。超量使用**不会**提高疗效，但会增加经皮吸收和全身不良反应风险。

**信息来源**：NMPA 批准说明书（蔓迪 5% 酊、达霏欣 2% 溶液）；StatPearls；Mayo Clinic；中华医学会科普

### 3.2 标准化喷涂/涂抹操作流程

#### 第一步：使用前准备

| 要求 | 说明 |
|------|------|
| 头皮状态 | **完全干燥、清洁**；头发与头皮均须干燥 |
| 适用区域 | 仅限**健康正常头皮**的脱发区域 |
| 不适用 | 头皮有炎症、感染、破损、/sunburn、瘢痕处 |
| 发型建议 | 男性建议短寸头便于药液接触头皮；女性需拨开头发暴露发缝 |

#### 第二步：精准喷涂/涂抹

**溶液/搽剂/喷雾剂标准喷涂法**（以 1 mL ≈ 7 喷为参考）：

1. 喷头**对准头皮**（非头发），距离适中，每喷**压到底**
2. 沿脱发区域**中线喷 3 下**，**两侧各 2 下**（可根据脱发范围调整，但总量不变）
3. 核心原则：**头皮湿、头发尽量不湿**——药液必须到达头皮表面
4. 从脱发区**中心**开始，向周边均匀覆盖
5. 如使用滴管：在脱发区分点滴涂，再均匀推开

**泡沫剂标准操作**：

1. 拨开头发，暴露脱发区
2. 罐体**倒置**，挤出**半瓶盖**泡沫于指尖
3. 手指将泡沫**均匀涂抹**于脱发区头皮

#### 第三步：轻柔处理（可选，见第四节）

- 用**指腹**轻轻按摩/叩拍 **3～5 分钟**（非必须）
- **禁止**用力抓挠、揉搓

#### 第四步：使用后处理

| 步骤 | 要求 |
|------|------|
| 洗手 | **立即**用肥皂和清水彻底清洗双手 |
| 避免接触 | 防止药液沾染面部、颈部、手臂等非目标区域 |
| 干燥等待 | 自然晾干 **2～4 小时**；**勿用吹风机**吹 scalp（降低吸收） |
| 洗头限制 | 用药后 **4 小时内避免洗头/游泳/大量出汗** |
| 就寝 | 确保药液干燥后再入睡，避免沾染枕套 |

**信息来源**：NMPA 说明书；Mayo Clinic；StatPearls；中华医学会科普；新加坡 National Skin Centre

### 3.3 喷涂区域范围

| 区域 | 是否推荐 |
|------|----------|
| 额颞部（M型脱发区） | ✅ |
| 头顶/旋涡区（O型脱发区） | ✅ |
| 发缝稀疏区（女性） | ✅ |
| 面部、眉毛、身体其他部位 | ❌ 禁止（致多毛症） |
| 有瘢痕/损伤的头皮 | ❌ 无效且增加吸收风险 |

---

### 3.4 使用时机与频次规范

| 项目 | 建议 |
|------|------|
| **最佳时间** | 固定 **早晚各一次**（2% 及男性 5% 溶液）；或 **每日一次**（女性 5% 泡沫/5% 溶液） |
| **间隔** | 两次用药间隔约 **12 小时** |
| **疗程** | 至少连续 **4 个月** 再评估；建议坚持 **1 年以上**；需**无限期维持** |
| **疗效评估节点** | 每 **3 个月** 评估一次（中国指南）；**1 年**后综合判断 |
| **漏用处理** | 想起则补用；若距下次不足4小时则跳过；**禁止**双倍剂量 |

---

### 3.5 喷涂前后的头皮护理要求

#### 使用前

- ✅ 头皮清洁、干燥即可；无需特殊"打底"产品
- ❌ 避免在**潮湿头皮**上用药（稀释药效）
- ❌ 避免在**刚洗完头未干**时使用
- ❌ 头皮有/active 皮炎、/sunburn 时暂停，待恢复后再用

#### 使用后

- ✅ 保持 **4 小时** 内不洗头、不游泳、不做大量出汗运动
- ✅ 立即洗手，避免面部接触
- ❌ 不要在用药区域**叠加**其他外用产品（同区域）
- ❌ 不要用热风吹干用药区域

#### 与美发化学处理的间隔

| 处理类型 | 要求 |
|----------|------|
| 染发、烫发、拉直 | 可以同时进行米诺地尔治疗，但：**处理前须先清洗 scalp**；米诺地尔需在处理前 **24 小时** 及处理后 **24 小时** 停用 |
| 普通洗发 | 用药后至少间隔 **4 小时** |

**信息来源**：Mayo Clinic；NMPA 说明书

---

### 3.6 个体化剂量调整

| 情况 | 调整建议 |
|------|----------|
| 出现肢体多毛（上臂、腿毛增多） | 提示吸收较多，可减少喷数（如 7 喷→5 喷或 4 喷） |
| 仅头面部多毛 | 多为涂抹/喷涂方法不当，改进技术即可 |
| 严重头皮刺激 | 换用**无丙二醇**泡沫剂，或降低浓度/频次 |
| 4 个月无改善 | 复诊评估，考虑联合治疗或调整方案 |
| 6 个月无改善 | 中国指南：可基本判断单纯药物治疗效果有限 |

## 四、安全搭配的按摩方案与辅助方案

### 4.1 按摩时机与方法（合规要点）

#### 核心结论

> **头皮按摩并非使用米诺地尔的必要步骤**（StatPearls 明确指出 "Scalp massage is not required"）。  
> 如选择按摩，应在**喷涂/涂抹后**进行，而非喷涂前。

| 项目 | 规范 |
|------|------|
| **时机** | **用药后**立即进行（喷涂/涂抹后） |
| **间隔** | 与用药同步，无需额外等待 |
| **时长** | **3～5 分钟**（国内说明书及中华医学会推荐） |
| **频率** | 每次用药后均可轻柔处理，即每日 1～2 次 |
| **手法** | **指腹**轻柔按摩或"扣拍"，促进均匀分布与吸收 |
| **力度** | **轻-中等**，以舒适为度；**禁止**用力抓挠、拉扯 |

#### ⚠ 过度按摩的风险（病例报告警示）

- 案例：47岁男性，每次用药后进行 **10～15 分钟** 仔细按摩 → 2 个月后出现**结节性脆发症（Trichorrhexis Nodosa）**及断发
- 调整：将按摩时间缩短至 **3～5 分钟** 后，断发明显改善
- **结论**：长时间、较用力按摩可造成**机械性毛发损伤**，抵消治疗获益

**信息来源**：StatPearls；J Cosmet Dermatol 病例报告（2024）；中华医学会科普；NMPA 说明书

---

### 4.2 专业按摩手法要点

```
推荐手法（每次 3～5 分钟）
────────────────────────────────────────
① 五指指腹置于脱发区头皮，而非头发上
② 小范围画圈式轻柔按压（力度：按压后头皮微红即可，无疼痛感）
③ 从脱发区中心向周边均匀扩散药液
④ 可用指腹"扣拍"代替揉搓（中华医学会推荐"扣一扣"）
⑤ 避免：指甲刮擦、暴力揉搓、梳子用力刮头皮
────────────────────────────────────────
```

#### 适用按摩工具

| 工具 | 建议 |
|------|------|
| **指腹** | ✅ 首选，最可控 |
| **硅胶头皮按摩刷（软毛）** | ⚠ 仅极轻柔使用，避免硬质齿梳 |
| **金属/硬质按摩梳、滚轮** | ❌ 不推荐，易致牵引/断发 |
| **电动按摩仪（强震动）** | ❌ 不推荐与米诺地尔同区域同时使用 |

### 4.3 可安全搭配的辅助成分与产品

#### A. 联合药物治疗（有循证支持）

| 联合方案 | 证据 | 推荐级别 |
|----------|------|----------|
| **米诺地尔 + 非那雄胺（口服，男性）** | Meta 分析：疗效评分高 **0.80 分** vs 单用米诺地尔 | 强推荐 |
| **米诺地尔 + 低剂量口服非那雄胺/度他雄胺** | 国内外多项 RCT | 男性一线联合 |
| **米诺地尔 + 螺内酯（女性，有高雄激素证据）** | 指南推荐 | 特定人群 |
| **米诺地尔 + 微针（Microneedling）** | Meta 分析：评分高 **1.19 分** vs 单药 | 需在医生指导下 |
| **米诺地尔 + 低能量激光（LLLT）** | Meta 分析：评分高 **1.03 分** vs 单药 | 辅助治疗 |

**信息来源**：《2023中国临床实践指南》；2020 AGA 联合治疗 Meta 分析

#### B. 头皮护理产品搭配原则

| 产品类型 | 搭配建议 |
|----------|----------|
| **温和无硅/shampoo** | ✅ 可用；用药后 **4 小时外** 洗发 |
| **酮康唑洗剂** | ⚠ 有一定辅助证据（抗炎/抗雄激素），但 AGA 高质量证据有限；避免与米诺地尔**同时间、同区域**叠加 |
| **含水杨酸/煤焦油去屑洗剂** | ⚠ 可能增加刺激；建议与用药**间隔 ≥ 4 小时** |
| **保湿/舒缓 scalp 精华** | ⚠ 不含酒精、强酸、高浓度精油；与用药**间隔 ≥ 4 小时** |
| **含咖啡因/肽类/植物提取物生发精华** | ⚠ 证据等级低于米诺地尔；避免同区域同时涂抹，防止成分冲突或稀释 |

#### C. 营养补充（针对合并缺乏者）

| 补充剂 | 说明 |
|--------|------|
| 铁剂 | 女性 FPHL 合并铁蛋白低时可考虑 |
| 维生素 D | 缺乏时补充可能有辅助价值 |
| 生物素 | 仅对生物素缺乏者有意义；对 AGA 直接疗效证据不足 |

**信息来源**：FPHL therapeutic update, 2023；2023中国指南

### 4.4 禁忌搭配与成分冲突

| 禁忌/高风险搭配 | 原因 | 处理 |
|-----------------|------|------|
| **同一 scalp 区域叠加其他外用药物/护肤品** | 增加吸收/刺激/疗效干扰 | 同区域避免；需间隔 **≥ 4 小时** |
| **含高浓度酒精、强酸（AHA/BHA）、过氧化物产品** | 增加头皮刺激、/contact dermatitis | 分开使用，间隔 4 小时以上 |
| **含丙二醇的其他产品**（对 PG 过敏者） | 交叉刺激 | 换用**无 PG 泡沫剂** |
| **染发/烫发/拉直剂** | 化学刺激 | 前后各 **24 小时** 停用米诺地尔 |
| **热吹风直接吹用药区** | 降低药物吸收 | 用药后自然干燥 |
| **低剂量阿司匹林** | 可能抑制磺基转移酶，**降低米诺地尔疗效** | 长期使用者需告知医生 |
| **胍乙啶（Guanethidine）** | 可致严重直立性低血压 | 禁止联用 |
| **环孢素（系统）** | 可能加重多毛症 | 需监测 |
| ** inflamed/破损/scalp 上使用** | 显著增加系统吸收 | 待恢复后再用 |
| ** microneedling 自行深刺（>1.5mm 无监督）** | 增加系统吸收与感染风险 | 仅在有资质医疗机构进行 |

**信息来源**：Mayo Clinic；StatPearls；Drugs.com 专业专论；Bioon 米诺地尔安全使用指南

## 五、不良反应与注意事项

### 5.1 常见局部不良反应

| 不良反应 | 发生率/特点 | 应对方案 |
|----------|-------------|----------|
| **头皮瘙痒、干燥、脱屑** | 最常见；5% 高于 2% | 换泡沫剂；降低浓度/频次；使用温和洗发 |
| **接触性皮炎（刺激型）** | 较常见，多与**丙二醇/乙醇**有关 | 换**无 PG 泡沫剂**；暂停后恢复 |
| **接触性皮炎（过敏型）** | 较少；需**斑贴试验**鉴别 | PG 过敏→换泡沫；米诺地尔本身过敏→停用 |
| **局部灼痛、红斑** | 5% 溶液更常见 | 减少剂量；换剂型；严重时停药就医 |
| **毛囊炎** | 偶见 | 保持清洁；持续恶化需就医 |
| **多毛症（Hypertrichosis）** | 约 **5%** 使用者；女性/5% 更常见 | 改进涂抹技术；避免面部沾染；洗手；通常停药后 **4～5 个月** 消退 |
| **面部/手臂意外多毛** | 药液转移所致 | 修正用法；立即洗手 |
| **狂脱期（初期脱发增多）** | 2～6 周常见 | 正常现象，持续数周；勿擅自停药 |
| **药物沾染眼部** | 可致灼痛/刺激 | 大量冷水冲洗 |

**信息来源**：Drugs.com Monograph；PMC Contact Dermatitis 系统评价；NMPA 说明书；Bioon 安全指南

---

### 5.2 少见但需警惕的全身不良反应

| 症状 | 可能原因 | 处理 |
|------|----------|------|
| 胸痛、心悸、心动过速 | 经皮吸收过多 | **立即停药**，急诊/就医 |
| 头晕、晕厥 | 血管扩张 | 同上 |
| rapid 体重增加、手足水肿 | 水钠潴留（系统吸收） | 同上 |
| 低血压 | 超量/大面积破损皮肤使用 | 同上 |
| 婴儿多毛症 | 照护者用药后婴儿接触用药区 | 避免儿童接触用药 scalp |

> 正常按说明书使用，系统吸收约 1.4%，严重全身反应**罕见**；但**超量、破损皮肤、/sunburn  scalp** 会显著增加风险。

**信息来源**：StatPearls；Drugs.com；UK SmPC（Minoxidil 5% cutaneous spray）

---

### 5.3 特殊人群注意事项

| 人群 | 建议 |
|------|------|
| **女性** | 优先 2% 或 5% 泡沫（每日1次）；严格避免面部沾染；5% 溶液每日 ≤ 1 mL |
| **男性** | 5% 溶液/泡沫，每日 2 次；可剃短头发提高给药精度 |
| **老年人（>65岁）** | 说明书标注慎用；需医生评估 |
| **儿童（<18岁）** | 国内说明书多标注禁用/慎用 |
| **孕妇/哺乳期** | **禁用**（FDA 妊娠 C 类；可经乳汁排泄） |
| **心脏病/高血压患者** | 慎用；需监测；告知医生 |
| **对 PG 过敏者** | 使用无 PG 泡沫剂 |
| **对米诺地尔过敏者** | 禁用所有含米诺地尔制剂 |

---

### 5.4 储存与用药安全

- 室温、密闭、避光保存；远离火源（含乙醇，**易燃**）
- 泡沫剂：勿刺穿、焚烧气罐；勿近明火
- 置于儿童不能触及处
- 过期药物勿用

---

### 5.5 何时应就医

- 用药 **4 个月** 仍无改善
- 狂脱期持续超过 **2 周** 且进行性加重
- 严重/持续 scalp 刺激、水疱、渗液
- 出现胸痛、心悸、水肿、晕厥等全身症状
- 非 scalp 区域出现明显多毛
- 计划联合其他药物（非那雄胺、抗雄激素、口服米诺地尔等）

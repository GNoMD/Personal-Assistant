# 小白 7 天晚间器械训练计划

面向：刚开始想动起来、改善圆肩驼背、兼顾增肌入门。  
时间：每天 **19:00** 起，约 40～55 分钟。  
器械：卧推架、史密斯机、哈克深蹲、深蹲架、髋外展/腿内收、高位下拉/坐姿划船、龙门架。

## 通用原则

- 重量以能标准完成目标次数、力竭前还能 1～2 次为准，宁轻勿假借力。
- 组间休息约 60～90 秒；动作质量优先于重量。
- 每天都做短热身与收尾拉伸；推日也保留面拉，避免越练越圆肩。
- 卧推、深蹲务必使用安全杠/安全销，不熟可先用史密斯。

## 体态提示

- **圆肩**：多做面拉、划船；夹胸/卧推后补胸部开门拉伸。
- **驼背**：强调肩胛后夹与胸椎伸展，避免只练胸不练背。
- 有疼痛、眩晕立即停止；慢性伤病请先咨询医生或康复师。

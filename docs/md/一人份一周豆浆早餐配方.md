## 0. 本周要顾全的五件事（总览）

| 目标 | 每周怎么保证「顾全」 |
|------|----------------------|
| 护肾控尿酸 | 高嘌呤豆严格限量；浸泡倒水；**周四低杂豆**；**周日零高嘌呤杂豆**；痛风红肿有替换表 |
| 健脾养胃 | **山药+茯苓天天有**；炒白扁豆周二/四/六；莲子/芡实高频；温饮+早餐搭配固形物 |
| 养发 | 绿心豆（原黑豆）/黑米/黑芝麻/核桃/鹰嘴豆**分散在一周**，不扎堆一天猛补 |
| 安神 | 莲子、百合**几乎天天有**；只作早餐，睡前 2 小时不再喝 |
| 增肌减脂 | 鹰嘴豆/藜麦/燕麦+豆渣提升蛋白与饱腹；**早餐固定配鸡蛋**；燕麦偏多日照顾减脂；**若每天健身见第 7 节加强版** |

> **打底三项（每天都有）**：健脾（山药/茯苓）+ 安神（莲子或百合）+ 稳糖纤维（燕麦片），再叠加当天主主题。

## 1. 饮用与制作（补全）

| 项目 | 规定 |
|------|------|
| 定位 | **早餐主饮** |
| 可饮量 | **500–600ml / 天**，默认 **一次喝完** |
| 打浆 | 干料 **56g** + 水 **550ml**（约 1:10） |
| 时间 | 早餐（如 7:00–9:00） |
| 标准搭配 | 豆浆 + **水煮蛋 1 个**（增肌优先；胃舒服可再加半片全麦） |
| 顺序 | 先咬两口蛋 → 再喝豆浆，或边吃边喝；**禁止纯空腹猛灌** |
| 温度 | **约 40℃**；禁止冰镇 |
| 豆渣 | **保留** |
| 浸泡 | 豆类/杂粮 **必须**泡 ≥4h 并倒水；山药茯苓莲子百合芡实 **建议同样泡软并倒水** |
| 机器 | 打至细腻；颗粒粗可再打一轮（更易消化） |
| 睡前 | **睡前 2 小时禁止再喝** |
| 临时调整 | 仅胃胀烧心当天：可先喝一半，上午再温热喝完；**克数不改** |

### 前一晚备料（建议）

1. 按次日配方称好干料；**豆类/杂粮** 与 **药食同源**（山药干、茯苓、莲子、百合、芡实等）分开放。
2. **必须泡、且倒掉泡水**：绿心豆（原黑豆）、黄豆、红腰豆、赤小豆、鹰嘴豆、绿豆、炒白扁豆，以及红米/黑米/小米/糙米/藜麦等杂粮 —— 凉白开泡 **≥4 小时**，打浆前倒掉泡水（降嘌呤、减胀气）。**燕麦片** 凉白开泡 **2～4 小时** 即可（比整粒燕麦米易软，泡太久易糊）。
3. **建议泡软、一般也倒掉泡水**：山药干、茯苓、去芯莲子、干百合、芡实 —— 凉白开泡 **≥4 小时**（干货回软，更好打细）；倒水主要为去浮沫、杂质和过浓药味，**不是**为降嘌呤。
4. 黑芝麻、核桃、花生：**不用长时间浸泡**；洗净即可（核桃可略泡软，非必须）。
5. 早晨：倒掉泡水 → 沥干 → 一并打浆 → 煮开/按豆浆机程序 → 温热食用。

## 2. 一人份红线（完整）

> **红线**：下列上限不可突破；痛风红肿期见文末应急替换。

| 类别 | 上限 / 规则 |
|------|-------------|
| 油脂 | 黑芝麻 ≤3g/日；核桃 ≤3g/日；花生仅周六 ≤3g |
| 高嘌呤豆 | （绿心豆+黄豆+红腰豆）≤18g/日；绿心豆（原黑豆）≤7g；黄豆 ≤5g；红腰豆一周 **2 次**（周二、周五） |
| 精制杂粮 | （红米+黑米+小米+糙米）≤8g/日 |
| 藜麦 | ≤4g/日 |
| 炒白扁豆 | 仅 **周二、周四、周六** |
| 周日 | 无绿心豆/黄豆/红腰豆/花生/绿豆/赤小豆/鹰嘴豆（肾脏缓冲日） |
| 痛风红肿期 | 停绿心豆（原黑豆）、黄豆、红腰豆、花生；当天改用「应急替换」 |

### 5.2 各种食材如何保存

**通用原则**

- **未开封**：阴凉干燥、避光；袋口密封或放入密封罐
- **已开封**：尽量 **密封 + 干燥**；潮湿地区可加食品干燥剂（勿直接接触食材）
- **已浸泡待打浆**：泡好后 **当天早晨打完**；若临时推迟，沥干后冷藏密封，**尽量 12 小时内用完**，勿反复泡放
- **忌**：受潮发霉、生虫、有哈喇味（油脂氧化）的坚果/芝麻一律丢弃

| 食材 | 常温密封（阴凉干燥） | 冷藏 | 冷冻 | 开封后建议用完 | 备注 |
|------|----------------------|------|------|----------------|------|
| **山药干** | 首选 | 潮湿季节可 | 可（长期） | 3～6 个月 | 防潮防虫；结块、变色、霉味勿用 |
| **茯苓** | 首选 | 潮湿季节可 | 可 | 6～12 个月 | 极怕潮；粉状更易结块，务必干燥剂 |
| **去芯莲子** | 可 | 潮湿季节建议 | 可 | 3～6 个月 | 易生虫，密封要严；发黄发黑挑除 |
| **干百合** | 可 | 潮湿季节建议 | 可 | 3～6 个月 | 避光；受潮发黏勿用 |
| **芡实** | 可 | 潮湿季节建议 | 可 | 3～6 个月 | 同莲子，防虫密封 |
| **炒白扁豆** | 可 | 一般不必 | 可 | 3～6 个月 | 已炒相对耐存；仍要防潮 |
| **绿心豆**（原黑豆） | 可 | 一般不必 | 可 | 6～12 个月 | 即绿心黑豆；干燥密封；生虫整袋检查 |
| **黄豆** | 可 | 一般不必 | 可 | 6～12 个月 | 同上 |
| **红腰豆** | 可 | 一般不必 | 可 | 6～12 个月 | 同上 |
| **赤小豆** | 可 | 一般不必 | 可 | 6～12 个月 | 同上 |
| **鹰嘴豆** | 可 | 一般不必 | 可 | 6～12 个月 | 干豆耐存 |
| **绿豆** | 可 | 一般不必 | 可 | 6～12 个月 | 耐存；防潮即可 |
| **燕麦片** | 可 | 开封后更推荐 | 可 | **1～2 个月**（开封） | 选原味老式；脂肪略高，开封后易哈败，小罐分装 |
| **藜麦** | 可 | 开封后可 | 可 | 2～3 个月（开封） | 密封防潮；有酸味勿用 |
| **黑米** | 可 | 一般不必 | 可 | 3～6 个月 | 密封；防虫 |
| **红米** | 可 | 一般不必 | 可 | 3～6 个月 | 同上 |
| **小米** | 可 | 一般不必 | 可 | 3～6 个月 | 易生虫，密封+阴凉 |
| **糙米** | 可 | **开封后建议冷藏** | 可 | **1～2 个月**（开封） | 谷皮含油脂，开封后易哈败 |
| **黑芝麻** | 短期待 | **开封后建议冷藏** | 长期首选 | 冷藏 1～2 个月 / 冷冻 3～6 个月 | 油脂易氧化，小份密封 |
| **核桃** | 短期待 | **开封后建议冷藏** | 长期首选 | 冷藏约 1 个月 / 冷冻 3～6 个月 | 仁易哈喇；带壳稍耐存 |
| **花生** | 短期待 | **开封后建议冷藏** | 长期首选 | 冷藏约 1 个月 / 冷冻 3～6 个月 | 同坚果；发霉（黄曲霉风险）整包丢弃 |
| **陈皮**（可选） | 干燥密封 | 一般不必 | 不必 | 长期可存 | 越陈越燥要防潮；霉变勿用 |

### 5.3 家庭分装建议（方便每天称重）

1. 采购后按「一周用量」称出本周份，装入 **7 个密封小袋/小盒**（或豆类一罐、药食一罐、杂粮一罐、坚果冷藏一小盒）。
2. **黑芝麻、核桃、花生** 单独冷藏小盒，避免与干货抢潮气、也减少哈败。
3. 每晚只取次日份去浸泡；不要整周原料一起泡。

## 6. 全套饮用须知

### 控糖 / 减脂

全程不加糖、红枣、桂圆、玉米碎；保留豆渣；精制杂粮每日 ≤8g；早餐豆浆后尽量正常午餐，避免再喝甜饮。

### 高尿酸 / 护肾

杂豆（尤其绿心豆、黄豆、红腰豆）严格按表；浸泡倒掉泡水；周四、周日降负担；红肿期用替换表。

### 高血压倾向（预防向）

本方含茯苓、赤小豆等高钾向食材；日常仍需 **低盐饮食** 配合。

### 高血脂 / 减脂

燕麦片几乎天天有；黑芝麻/核桃/花生严格控量，不可加勺芝麻酱。

### 养发

绿心豆（原黑豆）、黑米、黑芝麻、核桃、鹰嘴豆分天出现；豆浆仅为食养搭配，**不替代任何药物治疗或医嘱**；若头皮有外用药，用药后数小时内是否适合大量出汗，以说明书/医生意见为准。

### 失眠 / 安神

莲子百合高频；**只作早餐**；睡前 2 小时不喝，减少夜尿打断睡眠。

### 肠胃不适 / 胃食管不适

食材泡软打细；白扁豆日可重点选用；胀气可加陈皮 1–2g；**建议配鸡蛋等同食**；禁止冰镇；禁止空腹整杯灌。

### 大便调整

- 干结：可在当日额度内略增燕麦片、黑芝麻（各 +1g，相应减芡实或莲子 1g）
- 易泻：减少糙米、藜麦、绿豆；周六去掉花生

## 7. 每天健身时，豆浆怎么调（加强版）

> **前提**：力量训练若安排在 **晚间（如 19:00）**，豆浆仍只作 **早餐**，不要改成练后奶昔猛灌；蛋白主要靠正餐补。  
> **红线不能破**：不要为了增肌把绿心豆/黄豆/红腰豆天天加满。

### 7.1 总原则（比「不健身」多做什么）

| 项目 | 不健身/轻活动 | **每天健身** |
|------|----------------|--------------|
| 豆浆干料底方 | 56g / 550ml | **仍用同一套七天底方**（先保证控尿酸、养胃） |
| 早餐鸡蛋 | 1 个 | **2 个**（或 1 个蛋 + 无糖酸奶 150–200ml） |
| 豆浆里加豆 | 按表 | **禁止**为增肌临时加高嘌呤豆超红线 |
| 练前 1–1.5h | — | 少量碳水道具：香蕉半根 / 全麦一小片（勿空腹练） |
| 练后 1h 内 | — | **正餐蛋白**：鸡胸/鱼/瘦肉/豆腐，约掌心大小；配蔬菜 |
| 水分 | 正常 | 全天足量饮水；豆浆不算运动补液 |

### 7.2 豆浆「微调」规则（仍在红线内）

只允许在底方上做 **±2～4g 等量替换**，且满足：

- 精制杂粮（红米+黑米+小米+糙米）合计仍 **≤8g**
- 藜麦仍 **≤4g**
- 黑芝麻 ≤3g、核桃 ≤3g
- 高嘌呤豆（绿心豆+黄豆+红腰豆）仍 **≤18g**，周日仍为 **0**

| 训练类型（参考晚间计划） | 豆浆怎么微调 | 早餐外怎么补 |
|--------------------------|--------------|--------------|
| **拉背 / 推胸 / 腿部**（消耗大） | 燕麦片 **+2g**，相应减少山药干或茯苓 **2g** | 鸡蛋 **2 个**；午餐蛋白略加量 |
| **主动恢复 / 轻量日** | **不改底方** | 鸡蛋 1～2 个均可；避免大吃大喝 |
| **周四（养胃低豆日）** | **不改底方、不加杂豆** | 蛋白全靠 **鸡蛋+午晚餐肉蛋奶豆制品** |
| **周日（无杂豆护肾日）** | **不改底方、不加任何杂豆** | 同上；这天增肌靠正餐，不靠豆浆加豆 |
| **周五（已是蛋白日）** | 可把鹰嘴豆视为当天蛋白主力，**不必再加绿心豆** | 鸡蛋 2 个即可拉高早餐蛋白 |

### 7.3 不建议的「健身豆浆」做法

- 每天把绿心豆/黄豆加到上限「多补蛋白」→ 易顶穿尿酸红线
- 练后把早餐豆浆留到晚上喝 → 易影响睡眠/夜尿，也打乱「只作早餐」
- 豆浆里加蛋白粉、芝麻酱、大量坚果 → 油脂或加工添加超标，且非本方案食材
- 空腹只喝豆浆就去练 → 肠胃与表现都差

### 7.4 一天时间示例（早餐豆浆 + 晚练）

| 时间 | 安排 |
|------|------|
| 7:30–8:30 | 底方豆浆 500–600ml + **鸡蛋 2 个** |
| 12:00–13:00 | 正常午餐（蛋白+菜+适量主食） |
| 17:30–18:00 | 练前小加餐（半根香蕉或少量主食） |
| 19:00–20:00 | 力量训练 |
| 20:15–21:00 | 练后晚餐（蛋白优先）；**不再喝豆浆** |

### 7.5 和底方目标的关系

每天健身时：**控尿酸 / 养胃 / 养发 / 安神** 仍靠七天轮换底方；**增肌** 主要靠「鸡蛋加倍 + 午晚蛋白 + 训练」，豆浆只提供一部分植物蛋白与早餐能量，不要指望单靠改豆浆配方完成增肌。

## 8. 增肌减脂：豆浆以外也要补全

豆浆早餐只能提供 **一部分植物蛋白 + 纤维**。要「尽量增肌减脂」，请同时做到：

| 餐次 | 建议（示例） |
|------|----------------|
| 早餐 | 本豆浆 500–600ml + **鸡蛋 1～2 个**（每天健身用 2 个） |
| 午餐 | 掌心大瘦肉/鱼/豆腐 + 蔬菜 + 适量主食 |
| 晚餐 | 偏清淡蛋白+菜；若当晚训练，把蛋白放在练后；睡前 3 小时勿过饱 |
| 训练 | 可每天力量或「六练一恢复」；若头皮有外用药，训练时段遵医嘱 |
| 体重管理 | 减脂进度以 **腰围、体脂、体态变化** 为主，勿过度节食 |

## 9. 应急替换表（补全特殊情况）

| 情况 | 怎么改（克数对等替换） |
|------|------------------------|
| 痛风红肿 | 去掉绿心豆/黄豆/红腰豆/花生；用山药干、茯苓、燕麦片补足当天 56g |

## 10. 一周水果搭配（与豆浆同目标、相辅相成）

> 对齐每日主题：控尿酸 · 健脾养胃 · 养发安神 · 稳糖轻体。  
> **一人份**；与豆浆、鸡蛋同作早餐或上午加餐；**整果，不榨汁、不加糖。**

### 10.1 统一红线

| 规则 | 说明 |
|------|------|
| 份量 | 约 **80～150g / 天**（约一只拳头内），勿用水果当正餐猛堆 |
| 形态 | **整果或切块**；禁止果汁、奶茶、果干当「水果份」 |
| 时间 | **水果派对**在下午茶时段（约 15:30／16:00）；也可早餐同食少量；睡前 2 小时不吃 |
| 控尿酸 | 少果糖冲击；优先蓝莓/苹果/梨/猕猴桃/橙子；整果小份、别拿水果顶正餐 |
| 养胃日 | **周四**避开酸橙、青柠、未催熟猕猴桃；用香蕉或蒸苹果 |
| 痛风红肿 | 水果照常可选苹果/梨/橙子；仍忌酒精与高果糖饮料 |

### 10.2 每日搭配（跟豆浆主题走）

| 日 | 豆浆主题 | 水果（一人份） | 为什么配 |
|----|----------|----------------|----------|
| **一** | 养发安神稳糖 | **蓝莓 80～100g**（约半杯） | 多酚抗氧化，搭黑米芝麻养发；低 GI 不冲稳糖 |
| **二** | 健脾固肾 | **苹果 1 个**（约 180g，可去皮） | 果胶护肠、易消化，搭白扁豆健脾不打架 |
| **三** | 利水轻体 | **梨 1 个**（约 180～200g） | 水分足、清口，呼应赤小豆茯苓利水；一人份也更好控量 |
| **四** | 四神养胃 | **香蕉 1 根**（约 100～120g，七成熟） | 温和护胃、少酸刺激；反酸日首选，勿换酸橙 |
| **五** | 蛋白增肌养发 | **猕猴桃 1 个**（约 80～100g） | 维 C 助植物蛋白日吸收感；搭鹰嘴豆养发不扎堆蓝莓 |
| **六** | 清暑健脾 | **桃或梨 1 个**（约 150～200g） | 清暑润燥，呼应绿豆清火；胃弱选梨 |
| **日** | 无杂豆护肾 | **橙子 1 个**（约 150～180g） | 一人一只好控量；维 C 足、耐放省钱；给肾脏缓冲日加分 |

### 10.3 怎么吃（和豆浆的关系）

1. **顺序**：先几口蛋 → 豆浆 → 水果；或水果夹在中间，**勿空腹只吃酸果再灌豆浆**。
2. **健身日**：水果份量可不变；练前若要用碳，优先方案里已有的「半根香蕉」；当天若已是周四香蕉，练前改半片全麦即可，避免一天两根香蕉堆糖。
3. **买不到时的替换**（同目标内）：
   - 蓝莓 ↔ 橙子（份量：半杯蓝莓 ↔ 1 个橙子）
   - 苹果 ↔ 梨（胃舒服时）
   - 猕猴桃 ↔ 橙子
   - 橙子 ↔ 蓝莓 / 苹果
   - 仍避免：果汁、荔枝龙眼大量、果干当零食猛吃；不选西瓜（一人份难控量）、樱桃/草莓（偏贵或不耐放）

### 10.4 一周采购参考（一人）

| 水果 | 约一周量 | 出现 |
|------|----------|------|
| 蓝莓 | 80～100g | 1 天 |
| 苹果 | 1 个 | 1 天 |
| 梨（或桃） | 1～2 个 | 三、六 |
| 香蕉 | 1 根 | 1 天（+ 健身练前备用半根） |
| 猕猴桃 | 1 个 | 1 天 |
| 橙子 | 1 个 | 1 天 |

### 10.5 水果保存方法（常温 / 冷藏）

**通用原则**

- **未切开**：按下表选常温或冷藏；避开暖气、阳光直射、密闭潮湿
- **已切开 / 去皮**：一律 **密封冷藏**，尽量 **当天吃完**，最长不超过 **24 小时**
- **忌**：发霉、漏汁发黏、酒味/酵味、表皮大面积褐变发黑 — 整份丢弃
- **乙烯注意**：香蕉、猕猴桃（催熟中）会加速其他水果变熟变坏，尽量 **分开存放**

| 水果 | 常温怎么放 | 常温大约多久 | 冷藏怎么放 | 冷藏大约多久 | 备注 |
|------|------------|--------------|------------|--------------|------|
| **蓝莓** | 不推荐久放 | 当天～1 天 | **首选**：原盒或透气盒，垫厨房纸吸潮，勿提前洗净 | **2～3 天**（开封） | 洗后更易坏，**吃前再洗**；软烂、发霉整盒检查 |
| **橙子** | 阴凉通风可 | **约 5～7 天** | 保鲜袋或蔬果盒 | **1～2 周** | 一人刚好一只；切开后密封冷藏当天吃；表皮发霉整颗丢弃 |
| **苹果** | 阴凉通风可 | **约 5～7 天**（凉爽季节更久） | 保鲜袋或蔬果盒，远离洋葱异味 | **1～2 周** | 一人份一周吃 1 个，常温或冷藏都行；切开后密封冷藏当天吃 |
| **梨** | 阴凉通风可 | **约 3～5 天**（硬梨更耐放） | 保鲜袋，避免挤压 | **约 1 周** | 偏硬可常温再放 1～2 天变软；碰伤处易烂要早吃 |
| **桃** | 硬桃常温催熟 | 催熟 **1～3 天**；软熟后尽快吃 | 软熟后冷藏减缓 | 软熟后冷藏 **2～3 天** | 硬桃先常温至略软再吃或再冷藏；易碰伤 |
| **香蕉** | **必须常温** | 青黄过渡约 **2～4 天**；全黄后 **1～2 天**内吃 | **不要整根冷藏**（皮会黑、风味变差） | — | 可与其他水果分开放；已熟可挂起或垫纸；切段若冷藏需密封且尽快吃 |
| **猕猴桃** | 硬果常温催熟 | 催熟约 **2～5 天**（触感略软即可） | 催熟变软后冷藏 | 软熟后冷藏 **3～5 天** | 想慢点熟：未软前可直接冷藏；与苹果/香蕉放一起会催熟更快 |

**采购与存放小建议**

1. **周初采购**：苹果、梨、橙子、硬猕猴桃、青黄香蕉可一起买；蓝莓 **临近要吃那天买** 或买当天就冷藏并标日子。
2. **香蕉、硬猕猴桃**：常温区；软熟猕猴桃转冷藏。
3. **浆果类（蓝莓）**：进门即冷藏，**不要洗了再存**。
4. **切开的瓜果**：只留一餐量在外，其余密封冷藏。

- 炒白扁豆仅二/四/六，一周 3 次：护胃又不天天重复

## 13. 练后轻夜宵（与健身搭配）

> **定义**：晚练约 **19:00～20:00**；练后正餐 **20:15～21:00** 仍是蛋白主力。  
> 本节「夜宵」= **练后还饿时的轻补充**，不是第二顿晚餐。  
> 默认 **无乳糖、无糖** 酸奶；与豆浆一样 **睡前 2 小时不再进食**。

### 13.1 统一红线

| 规则 | 说明 |
|------|------|
| 时间 | **20:15～20:45**，最晚 **21:00** 前吃完 |
| 份量 | 轻量 **60～240 千卡**；不饿就 **跳过**（尤其恢复日） |
| 优先 | **无乳糖无糖酸奶**、**水煮蛋**、少量 **整果** |
| 禁止 | 奶茶、烧烤、泡面、果干一把、蛋白粉猛灌、冰镇酸奶（养胃日） |
| 与豆浆 | 豆浆只作早餐；**不要把豆浆留到晚上当夜宵** |
| 练前 | 17:30 半根香蕉/少量主食；与练后香蕉 **不要一天叠两根** |

### 13.2 按当晚训练轮换（跟 plan 健身日走）

| 健身日 | 训练主题 | 练后轻夜宵（还饿才吃） | 蛋白约 | 为什么 |
|--------|----------|------------------------|--------|--------|
| **一** | 体态启动/拉背 | 酸奶 150g + 蓝莓 30～50g | 10～12g | 中等消耗，抗氧化不增负担 |
| **二** | 胸推 | 酸奶 150g + 水煮蛋 1 个 | 16～18g | 推拉日略补蛋白 |
| **三** | 腿部 | 酸奶 200g + 香蕉半根 | 12～14g | 腿日补糖原；练前已吃香蕉则改 **蛋** |
| **四** | 背厚/养胃 | **温**酸奶 150g（不加果） | 10～12g | 养胃日勿冰；下午已有香蕉，夜宵不再叠果；反酸改酸奶+蛋 |
| **五** | 胸腿综合 | 酸奶 200g + 水煮蛋 1 个 | 18～20g | 消耗最大，与周五蛋白豆浆呼应 |
| **六** | 主动恢复 | 酸奶 100～150g **或跳过** | 0～10g | 不硬补，给肠胃/尿酸减负 |
| **七** | 全身巩固 | 酸奶 150g + 猕猴桃 ½ 个 | 10～12g | 维 C + 蛋白，一周收尾 |

### 13.3 其他可选食材（替换用）

| 食材 | 适合场景 | 注意 |
|------|----------|------|
| **无乳糖无糖酸奶** | 大多数练后日 | 冷藏 2～3 天；养胃日提前回温 |
| **水煮蛋 / 蛋清** | 蛋白不够、不想加豆 | 与早餐鸡蛋合计，别一天过量 |
| **内酯豆腐 100～150g** | 不想吃蛋/奶 | 少盐、不麻辣 |
| **无乳糖低脂奶 200mL** | 乳糖友好时 | 不加糖；不算「灌豆浆」 |
| **半根香蕉** | 腿日/大重量日 | 与练前、周四香蕉派对错开 |
| **蓝莓/猕猴桃/橙子** | 搭配酸奶 | 与水果派对统筹采购 |

## 14. 简要结论

1. **当早餐**：56g 干料 + 550ml 水 → 500–600ml，一次喝完，固定配鸡蛋。
2. **五项目标全周都有安排**，用覆盖表自检；不是只突出单一功效。
3. **原文 21 种食材全部用上**（含红米）；可选陈皮备用。
4. **水果一人份整果轮换** + **练后轻夜宵按健身日轮换**；不榨汁、不靠夜宵替代正餐蛋白。
5. 增肌减脂靠「豆浆早餐 + 鸡蛋 + 水果派对 + 练后正餐 + 可选轻夜宵 + 训练」一起补全。
6. **每天健身**：七天豆浆底方不用大改；加强靠 **鸡蛋加倍 + 练前小碳 + 练后正餐蛋白**，大重量日最多燕麦片 +2g 微调。

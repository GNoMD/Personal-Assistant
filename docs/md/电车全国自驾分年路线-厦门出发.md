# 厦门出发 · 电车（Model Y）全国分年路线（精确到省市）

> 目标：内地 **31 个省级行政区**都到过；景点打卡～经典热门。  
> 原则：**先桩密、后桩疏**；每省给出 **城市顺序 + 建议天数**。

## 总览

| 项目 | 打卡版 | 经典版 |
|------|--------|--------|
| 有效出行天数 | 约 180～280 天 | 约 280～400 天 |
| 建议跨度 | 2～3 年 | 3～4 年 |
| 费用（已有车、一人） | 约 15～30 万 | 约 25～45 万 |
| 覆盖 | 31 省各至少 1～2 个核心市 | 每省主线城市走完 |

## 分阶段说明

### 阶段一｜闽粤桂琼（约 27～38 天）

1. **福建**：厦门（出发）→ 泉州 → 福州 → 武夷山  
2. **广东**：潮汕进省 → 惠州 → 深圳 → 广州（粤北可选）  
3. **广西**：桂林 → 阳朔 → 南宁 → 北海  
4. **海南**：海口东线 → 三亚环岛（过海查政策）

### 阶段二｜浙沪苏皖赣（约 26～40 天）

5. **浙江**：杭绍甬（+舟山或浙南选一）  
6. **上海**：市区地铁 + 青浦/崇明短驳  
7. **江苏**：苏锡宁扬（苏北可选另加）  
8. **安徽**：黄山 → 合肥  
9. **江西**：婺源 → 景德镇 → 庐山 → 南昌

### 阶段三｜湘鄂豫（约 19～28 天）

10. **湖南**：长沙 → 张家界（+凤凰可选）  
11. **湖北**：武汉 → 宜昌（恩施可选）  
12. **河南**：郑州 → 开封 → 洛阳

### 阶段四｜鲁京津冀晋（约 25～37 天）

13. **山东**：济南 → 泰安 → 青岛 → 烟台/威海  
14. **北京**：城区少开车 + 郊野选一  
15. **天津**：市区短停  
16. **河北**：秦皇岛或承德 → 正定/石家庄  
17. **山西**：大同 → 平遥 → 太原

### 阶段五｜辽吉黑（约 17～27 天，优先夏秋）

18. **辽宁**：沈阳 → 大连（+丹东可选）  
19. **吉林**：长春 → 吉林市 → 长白山  
20. **黑龙江**：哈尔滨为主（雪乡/漠河慎选）

### 阶段六｜渝川黔滇（约 30～49 天）

21. **重庆**：市区 + 武隆或酉阳  
22. **四川**：成都 → 乐山/峨眉 → 川西或九寨选一段  
23. **贵州**：贵阳 → 黄果树 → 黔东南  
24. **云南**：昆明 → 大理 → 丽江（+香格里拉或腾冲选一）

### 阶段七｜陕宁甘青蒙（约 29～47 天）

25. **陕西**：西安（+华山或延安）  
26. **宁夏**：银川 → 中卫  
27. **甘肃**：兰州 → 张掖 → 嘉峪关 → 敦煌  
28. **青海**：西宁 → 青海湖（+茶卡或祁连）  
29. **内蒙古**：呼包鄂（阿拉善慎选；呼伦贝尔另开夏窗）

### 阶段八｜新藏 + 飞返（约 27～50 天）

30. **新疆**：北疆主线（乌鲁木齐→赛里木→伊犁→喀纳斯）；南疆另开窗口  
31. **西藏**：拉萨 + 林芝或日喀则（珠峰慎选）  
- **返程**：乌鲁木齐/拉萨/成都飞厦门；不建议自驾开回

## 执行建议

1. 每年选 **1～2 个阶段**（约 30～60 天）。  
2. 表中「可选 / 选一 / 慎选」是伸缩钮：时间紧就删。  
3. 住能过夜充电的店，优先于多塞一个市。  
4. 费用大头：住宿 > 门票 ≈ 过路 > 电费。
